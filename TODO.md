# Важно
## Анти абьюз система

1. Диминишинг XP

2. Кулдауны на типы наград

3. Анти-копипаст
    - совпадение текста > 80%
    - одинаковые сообщения подряд
    → XP = 0

4. Принцип НЕ наказания
Ты не штрафуешь, ты:
    - снижаешь эффективность
    - информируешь мягко
    - не стыдишь публично


## поведение → награда → визуал

1. Первое сообщение на стриме (DONE)
2. Ответ другому зрителю
3. Возврат после перерыва
4. Серия стримов
5. Участие > X минут
6. Слово дня
